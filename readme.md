## 🚀 Features

- ✅ Add, edit, and delete todos
- 🎨 Beautiful, modern UI
- 🔄 Real-time filtering (All, Active, Completed)
- 💾 Persistent storage with `localStorage`
- 🖱️ Interactive drag-and-drop (optional enhancement)
- ⚡ Super fast and lightweight
